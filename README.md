## 🎬 Overview

**SUic Player** is built on Google’s official [Media3 ExoPlayer](https://github.com/androidx/media), using both **Device Video Decoders** and **FFMPEG extensions** for maximum compatibility. It smoothly plays videos and audios, with all folders auto-discovered.

## 📁 Supported Formats

### 🔊 Audio
> Vorbis, Opus, FLAC, ALAC, PCM/WAVE, MP1, MP2, MP3, AMR (NB/WB), AAC (LC/HE/ELD/xHE), AC-3, E-AC-3, DTS, TrueHD, etc.

### 🎞️ Video
> H.263, H.264 AVC, H.265 HEVC, MPEG-4 SP, VP8, VP9, AV1

### 📦 Containers
> MP4, MOV, WebM, MKV, Ogg, TS, PS, FLV

### 🌐 Streaming
> DASH, HLS, SmoothStreaming, RTSP, HTTP/S

### 📝 Subtitles
> SRT, SSA, TTML, VTT

## 🌟 Key Features

- 📶 Multi Quality Streams  
- ⏩ Gesture Controls (Seek, Volume, Brightness)
